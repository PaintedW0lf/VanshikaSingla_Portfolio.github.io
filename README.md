Modern, interactive portfolio showcasing expertise in Software Engineering, Data Engineering, Data Analytics, and Leadership.

## Features

- **Interactive Elements**: Animated pet companion, floating sparkles, and decorative vines
- **Smooth Animations**: Scroll-triggered animations and hover effects
- **Responsive Design**: Optimized for all screen sizes
- **Dark Theme**: Elegant black and gold color scheme with gradient accents
- **Dynamic Projects**: Showcase of 8+ technical projects with live links
- **Professional Experience**: Chronological timeline of roles at AWS, ESDC, BC Liquor, and more

## Structure

- `index.html` - Main landing page with all sections
- `project-detail.html` - Template for individual project pages
- `styles.css` - Main stylesheet with dark theme and animations
- `project-detail.css` - Styles for project detail pages
- `script.js` - Interactive animations and pet companion logic
- `images/` - Project screenshots and profile images

## Sections

1. **Hero** - Introduction with call-to-action buttons
2. **About** - Professional background, achievements, and personal philosophy
3. **Experience** - Work history with AWS, ESDC, BC Liquor, FORA, UBC, and Edu-tainment
4. **Projects** - Technical projects including AI, web development, data analysis, and more
5. **Contact** - Links to resume, email, LinkedIn, and GitHub

## Technologies Used

- HTML5
- CSS3 (Custom animations, gradients, flexbox, grid)
- Vanilla JavaScript
- Google Fonts (Inter)
